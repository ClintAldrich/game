About the Game

What is "4 Pics 1 Word"?

This is a simple and fun word puzzle game. You're shown 4 pictures that have something in common — your task is to guess the one word that links them all!

How to Play:

  • Observe the 4 images carefully.
  • Type the word that connects them in the input box.
  • Click "Check Answer" to see if you're right.
  • If you're stuck, click "Reveal Hint" or try "Give Up" to see the answer.

Project Info:

This game was created as part of a school project to demonstrate basic web development skills including HTML, CSS, and JavaScript.
